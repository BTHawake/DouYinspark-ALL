# 源代码部署说明

> 前提：已通过[配置生成器](配置生成器使用.md)获取到完整的 CONFIG JSON。

可部署在本地设备、云服务器、青龙面板等环境。

## 1. 克隆项目

```bash
git clone https://github.com/2061360308/DouYinSparkFlow.git
```

## 2. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

## 3. 填写配置

在项目根目录创建 `config.json`，将配置生成器输出的 CONFIG JSON 粘贴进去。

## 4. 运行

```bash
python main.py
```
